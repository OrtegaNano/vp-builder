## Prompt de generación

```
CONTEXTO: {intake + TODAS las secciones completadas}

TAREA: Headline definitiva + Supporting Pitch.

### PARTE 1: HEADLINE

Genera 3 opciones. Cada una debe:
- Máximo 20-25 palabras
- Unir: ESPECIFICIDAD DE NICHO + RESULTADO TANGIBLE + INMEDIATEZ
- Sonar como si SOLO pudiera ser de este producto para este sector
- Pasar el "test de intercambiabilidad": si cambias el nombre del sector y sigue funcionando, es mala

Estructura que funciona: "[Qué es] para [quién] — con [diferenciador] que [resultado] desde [cuándo]."

Para cada opción, una línea de por qué funciona.

### PARTE 2: SUPPORTING PITCH

2 párrafos que expandan la headline:

Párrafo 1 — Qué es y qué NO es:
"Esto no es [lo genérico]. No es [lo que ya probaron]. Es [lo que realmente es]. Cada [elemento del producto] está construido alrededor de [lo específico]: [2-3 ejemplos concretos de workflows]."

Párrafo 2 — El resultado:
"Tu [equipo/empresa] obtiene [resultado A] y [resultado B]. [La doble promesa en una frase]. El resultado: [frase de cierre potente]."

REGLAS:
- El headline se destila de todo el trabajo previo, no se inventa desde cero.
- Cada palabra debe tener peso. Si puedes quitarla sin perder significado, quítala.
- El Supporting Pitch debe hacer que alguien que solo lea esos 2 párrafos entienda completamente la propuesta de valor.
- No usar jerga vacía. No usar "revolucionario", "innovador", "de última generación".

FRAMEWORK: Fórmula de headline = Especificidad + Resultado + Inmediatez (inspirado en David Ogilvy: "be specific, be concrete, be immediate").
```
